## Pitfalls

1. **FTS5关键词匹配** - engram使用FTS5，不是语义搜索。中文查询可能找不到英文记忆。
2. **engram需要正确工作目录** - 必须cd到git repo目录再调用。
3. **web_search token过期** - Firecrawl token会过期，用open-websearch替代。
4. **memory replace需要短唯一子串** - 不要用含转义字符的完整文本。
